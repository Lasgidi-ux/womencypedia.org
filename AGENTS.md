# Agent Commands Reference

This file contains command references for development tasks.

## Code Quality
- **Linting**: `npm run lint` (not currently configured - add ESLint if needed)
- **Type Checking**: `npm run typecheck` (not currently configured - add TypeScript if needed)

## Build Commands
- **Full Build**: `npm run build`
- **CSS Build**: `npm run build:css:prod`
- **JS Build**: `npm run build:js`

## Development
- **Dev Server**: `npm run dev`
- **Watch CSS**: `npm run watch:css`
